# TODO LIST アプリ

Udemyの講座「5つのアプリ作成とファイルのバンドリング〜JavaScriptのプログラミング」で学習した成果物です。

## デモ
👉 [アプリを試す](https://takii-252.github.io/udemy-js-webpack/src/)

## 機能
- TODOの追加
- TODOの完了/未完了の切り替え
- TODOの編集
- TODOの削除
- 検索によるフィルタリング
- ローカルストレージへのデータ保存（ブラウザを閉じてもデータが残ります）

## 使用技術
- HTML
- CSS
- JavaScript (ES6 modules)

## 学習メモ
コード内に日本語の注釈を追加しており、各処理の意味を確認できるようにしています。
